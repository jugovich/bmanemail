main — main¶

main.build_model(training_file, model_dir)[source]¶

Creates a C4.5 classification model (weka J48 classifier) using a supplied arff training file and outputs the completed model to supplied model_dir. The weka.classifiers.trees.J48 is called using pythons built-in subprocess module by calling the java process + a list of command line arguements. See http://docs.python.org/library/subprocess.html

main.classify(test_file, model_file, save_file, predict_var_name)[source]¶

Apply a C4.5 classification model to the test_file using supplied model_file, to predict variable at index of predict_var_name. The predictions are captured using subprocess.check_output See http://docs.python.org/library/subprocess.html, These predicitons are parsed and then stored to an output file (save_file). The output file contains the processed dataset information with predicted values and path_id appended.

main.gen_sample(data_path, save_path, sample)[source]¶

This module takes a dataset (data_path) and saves a sample of it to a new dataset (save_path). This module was used to gernate the training and validation samples for the ect584 project.

main.run_clean(data_dir, save_dir, filename, format)[source]¶

This module cleans a file (filename, format) at data_dir. This cleaning is necessary to remove unwanted information before running process(). The file is saved to save_dir.

main.run_extract(data_dir, save_dir, filename, format, verbose)[source]¶

This module takes a data directory pointing where the raw email files are stored in seperate directories (one directory for each list), takes a save_dir, filename, and format for saving the extracted raw dataset. The module loops over all the email list directories and loops over all the email files in these directories calling bmine.extract() on each email. These emails are added to a raw_extract dataset. Specify verbose to see all extraction details (this will slow down processing time).

main.run_process(data_dir, save_dir, filename, format)[source]¶

This module loads a dataset and processes it to produces a processed trainining dataset and test dataset in arff format to be used to create a classification model and then apply this module to the test dataset.

main.run_review_terms()[source]¶

This module was used to review the terms present in the raw/cleaned datafiles.

main.run_singleExtract(data_dir, save_dir)[source]¶

This module can extract a single raw directory, (used for testing purposes).

main.stage_train(data_dir, data_filename, data_format, train_dir, train_filename, train_format, save_dir)[source]¶

This module takes a test dataset (data_dir, data_filename, data_format) and adds the predicted event values in the coded training dataset (train_dir, train_filename, train_format) and saves the test dataset with predicted values added to save_dir.

main.validate(data_dir, filename)[source]¶

This module compares a dataset that has predicted values to the validation dataset and then outputs statistics on the accuracy of the model.
